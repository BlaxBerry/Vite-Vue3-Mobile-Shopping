## Vite-Vue3-Mobile-Shopping
SPA Mobile EC shopping web application.

- rewrote from:
    **Vue2 + Vue-Cli** + Webpack 
- to: 
    **Vue3 + Vite**


### Tech Stacks
- Vite2
- Vue3
- VueX4
- Vue-Router
- Vant ( UI Components )
- Less
- (Rem Unit)
    - postcss
    - amfe-flexible
- Axios
