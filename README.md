## Vite-Vue3-Mobile-Shopping

SPA Mobile EC shopping web application.

### Tech Stacks

- **Vite2**
- **Vue3**
- Vuex4
- Vue-Router4
- **Vant3** ( UI Components )
- Less
- postcss
- amfe-flexible
- Axios
